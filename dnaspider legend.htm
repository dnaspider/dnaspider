dnaspider legend (v2.2.4):

make sure tips in main tab is checked, press backspace, then double click blank text box in db tab to get started

or in the blank text box,
input dna > i/o code followed by an algorithm. (when you press a key on your keyboard you get a dna > i/o code in dna program title bar. right click main > tips for options)
ctrl + s to add it to the database. 
now using your keyboard output dna > code to run custom algorithm. 

//example		//dna >	i/o code (4)	//algorithm

code length basic:	test			«bs*4»Hi!«ctrl»a«-ctrl»
with note:		test			«//print Hi! and select it»«bs*4»Hi!«shift»«left*3»«-shift»
no code length:		«tests»			«bs*5»Hi 
no code length2:	«tests-»		Yeah Buddy

basic2:			test			¤¤¤¤Hi!•aÞ
(ultimately your textbox will look like this before you ctrl+s to save item to database listbox: «tests-»Yeah Buddy!)

tip: 
uncheck main > tips for more direct editing (no popup input boxes | textbox -> listbox )
set tabs > main > length options to 4. this is your dna > i/o code length.
next, go to tabs > ignore > & uncheck options you will be using for your custom dna > i/o keyboard shortcut code.
when ready, press 4 keys then (input example: Ctrl Esc Esc Esc)
go to tabs > db and double click blank text box; (output example: C...)
this is your dna > i/o code. right after your code type in a 
message/algorithm (right click for algorithm code menu). 
ctrl + s to add it to the database. finally, type the custom 4 key code in the text box for results.
example: C...hello
